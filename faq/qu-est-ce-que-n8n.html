Qu'est-ce que n8n ?

n8n est un outil open-source d'automatisation de workflows, alternative européenne à Zapier et Make. Il permet de connecter plus de 500 applications (CRM, email, téléphonie, IA) via une interface visuelle. Auto-hébergeable gratuitement, ou cloud à 20€/mois.

Détail de la réponse

n8n (prononcé 'n-eight-n') est une plateforme d'automatisation de workflows open source développée en Allemagne. Elle permet de créer des automatisations entre différentes applications sans coder, via une interface visuelle à base de nœuds connectés. n8n supporte plus de 500 intégrations natives (HubSpot, Gmail, Slack, Twilio, OpenAI, etc.) et permet aussi d'écrire du code JavaScript custom pour des logiques avancées. Ses avantages principaux : auto-hébergeable gratuitement (Docker), données en Europe (RGPD), beaucoup plus puissant que Zapier, et prix imbattable (20€/mois pour la version cloud). C'est l'outil de référence en 2026 pour les PME qui veulent automatiser sérieusement sans exploser leur budget.
